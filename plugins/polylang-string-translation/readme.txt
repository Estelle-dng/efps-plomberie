Parse template and generate Polylang String Translation Config file.

== Description ==

This plugin automatically generate the PHP file with all your string translation inside your theme.
The generated file contain all of your theme string using Polylang function "pll_register_string". Just use the WordPress API functions.

Require Polylang v 0.6 minimum.

== Installation ==

Use WordPress' Add New Plugin feature, searching "Polylang String Translation", or download the archive and:

1. Unzip the archive on your computer  
2. Upload `polylang-string-translation` directory to the `/wp-content/plugins/` directory
3. Activate the plugin through the 'Plugins' menu in WordPress
4. Go to Settings -> Translation and generate your config file
